#  JavaScript Quiz App (Permanent Dark Mode)

A responsive and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.  
This project features a **permanent dark-themed UI**, multiple-choice questions, a countdown timer, score tracking, and quiz restart functionality.

---

##  Project Objective

To gain advanced skills in:
- Responsive web design using CSS
- JavaScript DOM manipulation
- Building interactive UI components
- Managing application state (questions, timer, score)

---

##  Features

-  **Permanent Night Mode UI**
-  **Countdown Timer for each question**
-  **Multiple-choice quiz questions**
-  **Instant correct / wrong answer feedback**
-  **Final score display**
-  **Restart quiz option**
- **Fully responsive (mobile, tablet, desktop)**
